选题与开题

1. 选题方向探索

我是一名[专业/年级]学生，需要写一篇[课程论文/毕业论文]。
我感兴趣的领域是：[填写1-2个关键词]
请帮我：
1. 在这个领域内列出10个可能的研究方向
2. 每个方向标注：研究热度（高/中/低）、资料可得性（容易/一般/困难）、创新空间（大/中/小）
3. 推荐3个最值得做的方向，说明理由

2. 选题可行性评估

请评估以下论文选题的可行性：
- 选题：[填写具体题目]
- 我的条件：[专业、年级、可用的数据库/工具、时间周期]
- 要求：
  1. 这个选题有哪些潜在的研究难点？
  2. 需要用到哪些研究方法？
  3. 大概需要什么类型的数据？能否获取？
  4. 如果不可行，请推荐3个调整方向

文献综述

3. 文献综述框架

请为以下主题设计一份文献综述的写作框架：
- 论文主题：[填写]
- 已有文献数量：约[XX]篇
- 要求：
  1. 分成3-5个小节（按主题/方法/观点划分皆可）
  2. 每小节标注"核心论点和代表性文献"
  3. 最后设一节"研究述评"：总结已有研究的贡献 + 指出不足 + 引出本研究

4. 单篇文献精读

请帮我精读这篇论文：
- 标题+摘要：[粘贴]
- 要求：
  1. 核心论点：作者的核心观点/结论是什么？
  2. 研究方法：用了什么数据/方法？样本量？分析方法？
  3. 理论贡献：在已有研究基础上推进了什么？
  4. 可借鉴之处：我可以在哪些方面引用/借鉴这篇论文？
  5. 不足之处：这篇论文有什么局限？

正文写作与润色

5. 段落扩写

请将以下要点扩写成完整的学术段落：
- 核心观点：[一句话概括]
- 支撑论据：[列出2-3个论据/数据/引用]
- 要求：
  1. 学术语言，但避免过度使用"笔者""本文"等
  2. 论据之间有逻辑连接词
  3. 段落长度200-400字
  4. 结尾句要承上启下

6. 降重改写

请帮我改写以下段落以降低重复率：
- 原文：[粘贴]
- 要求：
  1. 保持学术原意不变
  2. 不要只是同义词替换，要改变句子结构
  3. 可以合并或拆分句子
  4. 不要引入原文没有的信息或观点
  5. 提供3个不同改写版本，标注"改变策略"（句式重组/信息重构/视角转换）
