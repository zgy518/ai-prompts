第一章：选题与开题

方法论：好选题的三个标准

1. 有资料来源 — 能找到足够文献支撑（至少20篇核心文献）

2. 有研究空间 — 不是被写烂的话题，有一定创新角度

3. 有能力完成 — 数据可获取、方法你掌握、篇幅可控

AI在选题阶段的作用不是"帮你想题目"，而是帮你缩小范围、验证可行性、发现研究空白。

1. 选题方向探索 我是一名[专业/年级]学生，需要写一篇[课程论文/毕业论文]。 我感兴趣的领域是：[填写1-2个关键词] 请帮我： 1. 在这个领域内列出10个可能的研究方向 2. 每个方向标注：研究热度（高/中/低）、资料可得性（容易/一般/困难）、创新空间（大/中/小） 3. 推荐3个最值得做的方向，说明理由
2. 选题可行性评估 请评估以下论文选题的可行性： - 选题：[填写具体题目] - 我的条件：[专业、年级、可用的数据库/工具、时间周期] - 要求： 1. 这个选题有哪些潜在的研究难点？ 2. 需要用到哪些研究方法？ 3. 大概需要什么类型的数据？能否获取？ 4. 如果不可行，请推荐3个调整方向
3. 研究问题提炼 我想研究的主题是：[填写] 请帮我把这个"主题"转化为具体可研究的"问题"： 1. 列出3-5个可能的研究子问题 2. 每个子问题应该满足：可验证、有边界、有学术意义 3. 用"X对Y的影响机制研究——以Z为例"的格式，给出3个论文题目建议
4. 开题报告框架 请帮我生成一份[本科/硕士]论文开题报告框架： - 论文题目：[填写] - 要求包含以下部分（每部分给出写作提示）： 1. 选题背景与意义（为什么研究这个） 2. 国内外研究现状（简述已有研究 + 指出现有不足） 3. 研究内容与方法 4. 创新点与预期成果 5. 研究进度安排 - 每部分给2-3句话的"写作方向提示"
5. 研究背景撰写 请帮我撰写论文第一章"研究背景"部分： - 论文主题：[填写] - 要求： 1. 从宏观到微观，逐层聚焦到研究问题 2. 引用2-3个权威数据或政策背景 3. 最终落脚到"本研究要解决什么问题" 4. 字数800-1000字 5. 学术语言但不要过度晦涩
6. 理论框架选择 我的论文研究方向是：[填写] 请帮我： 1. 推荐3-5个可能适用的理论框架 2. 每个理论简要说明：核心观点、代表人物、为什么适用于我的研究 3. 推荐最合适的一个，并说明如何将其应用到我的分析中 注意：如果是本科论文，优先推荐经典、成熟的理论
7. 研究假设构建 基于以下研究问题：[填写] 请帮我构建3-5个可检验的研究假设： - 每个假设应包含：自变量、因变量、预测的关系方向 - 为每个假设提供：理论基础（为什么会有这个预测） - 格式：H1: [自变量]对[因变量]有[正向/负向]影响，因为[理论依据]
8. 创新点提炼 结合以下论文信息，请帮我提炼2-3个创新点： - 研究主题：[填写] - 研究方法：[填写] - 研究对象/数据：[填写] - 要求： 1. 创新点要具体，不能说"选题新颖"这种套话 2. 每个创新点用"与已有研究不同，本研究……"的句式 3. 区分"理论创新"和"方法创新"

第二章：文献检索与综述

方法论：文献综述不是文献罗列

很多人写文献综述就是"A说了X，B说了Y，C说了Z"。这不是综述，是目录。

好的文献综述要做到：归类（按主题/方法分组）→ 比较（指出异同和争议）→ 批判（指出不足=你的研究空间）→ 衔接（自然引出你的研究）

9. 关键词扩展 我的论文主题是：[填写] 请帮我扩展检索关键词： 1. 中文关键词：列出10-15个（包括同义词、上下位概念、相关理论名称） 2. 英文关键词：列出对应的10-15个英文关键词 3. 推荐检索式：用AND/OR组合写出3条可用于知网/Web of Science的检索式
10. 文献分类整理 请帮我将以下文献进行分类整理： - 文献列表：[粘贴论文标题+摘要] - 要求： 1. 按研究主题分成3-5个类别 2. 每个类别下按时间顺序列出文献 3. 标注每篇文献的核心观点（一句话） 4. 指出不同类别之间的交叉和争议点
11. 文献综述框架 请为以下主题设计一份文献综述的写作框架： - 论文主题：[填写] - 已有文献数量：约[XX]篇 - 要求： 1. 分成3-5个小节（按主题/方法/观点划分皆可） 2. 每小节标注"核心论点和代表性文献" 3. 最后设一节"研究述评"：总结已有研究的贡献 + 指出不足 + 引出本研究
12. 单篇文献精读 请帮我精读这篇论文： - 标题+摘要：[粘贴] - 要求： 1. 核心论点：作者的核心观点/结论是什么？ 2. 研究方法：用了什么数据/方法？样本量？分析方法？ 3. 理论贡献：在已有研究基础上推进了什么？ 4. 可借鉴之处：我可以在哪些方面引用/借鉴这篇论文？ 5. 不足之处：这篇论文有什么局限？
13. 文献综述段落撰写 请将以下文献信息整合成一段文献综述： - 主题：[填写] - 相关文献：[粘贴3-8篇文献的核心观点] - 要求： 1. 先概括这一研究方向的整体情况 2. 有逻辑地串联各文献的观点（不是罗列） 3. 指出这些研究之间的共识和分歧 4. 最后指出："然而，现有研究在……方面仍存在不足" 5. 字数300-500字
14. 参考文献格式 请将以下文献信息转换为[GB/T 7714 / APA / MLA]标准格式： - 文献信息：[粘贴标题、作者、期刊、年份、卷期、页码等] - 确保格式完全符合标准要求 - 如果信息有缺失，标注[信息缺失]并说明缺什么
15. 研究空白识别 基于以下文献综述内容：[粘贴你的综述草稿] 请帮我识别研究空白： 1. 目前研究主要集中在哪些方面？ 2. 哪些角度还没被充分研究？ 3. 哪些结论存在争议或矛盾？ 4. 推荐2-3个有潜力的研究切入点
16. 国内外研究对比 请帮我对比[研究主题]的国内外研究现状： - 要求用表格呈现： | 维度 | 国内研究 | 国外研究 | | 研究重点 | ... | ... | | 常用方法 | ... | ... | | 理论框架 | ... | ... | | 主要结论 | ... | ... | | 不足之处 | ... | ... | - 最后总结：国内研究可以向国外借鉴什么？国外研究忽视了什么中国特色？

第三章：论文大纲设计

方法论：大纲是论文的骨架

论文不是按"想到的顺序"写的，而是按"逻辑的顺序"

每个章节只回答一个问题。如果一章回答了两个问题，拆成两章

章节之间有递进关系，不能随意调换顺序

标题要见题知意，不要用"相关理论"这种模糊标题

17. 整体大纲生成 请帮我设计一份论文大纲： - 论文题目：[填写] - 论文章节数量：[5-7章] - 要求： 1. 每章给出标题 + 2-3句话的"本章要回答的核心问题" 2. 每章下列出3-5个小节标题 3. 标注各章之间的逻辑关系 4. 估算每章建议字数（以总字数[X万字]为基准）
18. 章节逻辑检查 请帮我检查这份论文大纲的逻辑是否合理： - 大纲：[粘贴] - 检查要点：章节顺序是否自然递进？有没有重复内容？有没有遗漏的关键环节？标题是否准确反映内容？ - 如有问题，请给出具体修改建议
19. 某章节详细展开 我的论文大纲如下：[粘贴大纲] 请将第[X]章"[章节名]"展开为详细写作提纲： 1. 分为4-6个小节 2. 每小节写：核心论点 + 支撑材料 + 预计字数 3. 小节之间要有逻辑递进
20. 理论框架章节设计 我的论文需要写一章理论基础： - 涉及的理论：[填写1-3个理论名称] - 要求：每个理论写清提出者/年代/核心观点/核心概念，说明与本研究的关系，构建理论分析框架图（文字描述+ASCII图示），字数3000-5000字
21. 对比分析型论文大纲 我的论文是比较研究：[对象A]与[对象B]的对比。请设计比较型论文大纲： - A方案（先分后合）：分别介绍A和B，最后对比 - B方案（维度假设）：按对比维度分章，每章同时对比A和B - 推荐最适合我选题的结构，并给出理由
22. 案例分析型论文大纲 我的论文是案例分析：[案例名称] 请设计案例分析型论文大纲： - 结构：引言→文献综述→案例描述→案例分析→讨论→结论 - 每项标注具体内容，案例分析部分标注分析维度

第四章：正文写作

方法论：先完成再完美

正文写作最大的陷阱是"边写边改"。很多人在第一段反复打磨，三天过去了还停在引言。

正确策略：快速出草稿（AI辅助）→ 冷处理（放一天）→ 分轮修改（逻辑→表达→细节）→ AI润色

23. 引言撰写 请帮我撰写论文引言部分： - 论文题目：[填写] - 研究背景关键信息：[填写2-3条] - 要求： 1. 第一段：现实/政策背景，引出研究主题 2. 第二段：简要提及已有研究，指出不足 3. 第三段：说明本研究要做什么、用什么方法 4. 第四段：概述论文结构 5. 总字数800-1200字，每段之间要有过渡句
24. 段落扩写 请将以下要点扩写成完整的学术段落： - 核心观点：[一句话概括] - 支撑论据：[列出2-3个论据/数据/引用] - 要求：学术语言但避免套话，论据之间有逻辑连接词，段落长度200-400字，结尾句承上启下
25. 数据分析描述 请帮我撰写数据分析部分： - 分析方法：[填写：描述性统计/回归分析/内容分析等] - 关键发现：[填写3-5条核心数据结果] - 要求：先说明方法再呈现结果，不要只重复数字要解释"这意味着什么"，重要结果在段首标注，不显著的结果也要如实报告
26. 论证展开（三段论） 请用"观点-论据-分析"三段论结构撰写以下内容： - 观点：[填写] - 可用论据：[填写] - 要求：先明确提出观点→给出论据支撑→分析论据如何支持观点（最关键）→回应可能的反驳
27. 案例描述 请帮我撰写案例描述部分： - 案例背景：[填写] - 要求：客观描述不加评价（评价放分析部分），按时间线或逻辑线组织，突出与研究问题相关的信息，字数500-800字
28. 讨论部分撰写 请帮我撰写论文讨论部分： - 主要发现：[列出3-5条] - 已有研究结论：[列出相关文献核心发现] - 要求：逐一讨论你的发现与已有研究的关系，如果一致解释为什么，如果不一致分析可能原因，讨论理论贡献和实践启示，坦诚说明研究局限
29. 结论撰写 请帮我撰写论文结论部分： - 研究问题：[填写] - 主要发现：[填写3-5条] - 要求：先回顾研究问题 → 总结核心发现（凝练，非重复） → 说明理论贡献和实践启示 → 指出研究局限和未来方向 → 最后一段有"收束感" → 总字数600-900字
30. 过渡段撰写 请为以下两个章节/段落写过渡段： - 上一部分的内容：[一句话概括] - 下一部分要写：[一句话概括] - 要求：先简要总结前文 → 指出前文的"留白" → 自然引出下文 → 2-4句话即可
31. 摘要撰写 请帮我撰写论文摘要： - 研究背景+目的+方法+主要发现+结论与启示各用1-2句话 - 总字数200-300字，不出现"本文""笔者"，关键词3-5个
32. 引言-文献-结论联动检查 我的论文有三部分内容如下，请检查是否逻辑闭环： - 引言提出的研究问题：[填写] - 文献综述指出的研究空白：[填写] - 结论回答的问题：[填写] - 检查：引言提的问题结论有没有回答？文献综述指出的空白有没有填补？如果没有闭环，指出断裂点

第五章：润色与降重

方法论：降重的本质是"消化再表达"

无效操作：A词换B词、主动句改被动句（在智能查重面前基本无效）

有效策略：理解后用自己的话重写（最根本）→ 改变句式结构（句子重组）→ 合并或拆分段落（改变信息组织方式）→ 增加自己的分析（引用后多写"这意味着"）

33. 学术语言润色 请帮我润色以下段落，提升学术性： - 原文：[粘贴] - 要求：保持原意不变，提升用词准确性（"很多"→"大量"，"因为"→"归因于"），减少口语化表达，保持句子长度适中，给出修改版+修改说明
34. 降重改写 请帮我改写以下段落以降低重复率： - 原文：[粘贴] - 要求：保持学术原意不变，不要只是同义词替换要改变句子结构，可以合并或拆分句子，不要引入原文没有的信息或观点，提供3个不同改写版本并标注"改变策略"（句式重组/信息重构/视角转换）
35. 逻辑流畅度检查 请检查以下段落的逻辑流畅度： - 段落：[粘贴] - 检查：句间是否有逻辑跳跃？是否有重复啰嗦？连接词是否得当？段落是否有统一的中心思想？给出修改版
36. 长句拆分 以下长句过于复杂，请拆分： - 原句：[粘贴] - 要求：拆分为2-4个短句，保持逻辑关系不变，过渡自然
37. 避免学术套话 以下段落存在"学术套话"问题（空洞漂亮话），请帮我改写： - 原文：[粘贴] - 要求：删除"随着经济社会的发展""在新形势新背景下"等无信息量套话，删除"具有重要理论意义和现实意义"等无需论证的断言，用具体信息替代模糊表述，标注删了什么及原因
38. 摘要润色 请帮我润色论文摘要： - 摘要：[粘贴] - 要求：每个字都有信息量，第一句就说清"研究了什么发现了什么"，学术关键词密度要高，不出现"本文""本研究"以外的自我指涉词
39. 全篇一致性检查 请检查以下文本的一致性： - 文本：[粘贴] - 检查：术语是否前后一致？时态是否一致？人称是否一致（统一用"本文"或"本研究"）？中英文标点是否混用？数字格式是否统一？
40. 查重预检 以下段落我怀疑重复率较高，请用"重写"策略帮我降重： - 原文：[粘贴] - 要求：我先用我的理解把这段话说一遍[粘贴你的复述]，请基于我的理解版本润色成正式学术表达，确保不会与原文有连续13字以上的重复

第六章：答辩准备

方法论：答辩不是"汇报"，是"答辩"

评委已读过你的论文，不需要你再念一遍

陈述控制在5-8分钟，问答10-15分钟

PPT不是Word搬家，每页最多3条要点

准备比表现更重要：提前预判10个最可能被问到的问题

41. 答辩陈述稿 请帮我撰写答辩陈述稿： - 论文题目：[填写] - 答辩时长：[5/8/10分钟] - 要求：结构为研究背景(30秒)→研究问题(30秒)→研究方法(1分钟)→主要发现(2-3分钟)→创新点与贡献(1分钟)→不足与展望(30秒)，用口语化的学术语言（能说出来的话），每段标注时间，避免背诵感

附赠：学术Prompt进阶技巧

技巧1：角色设定的精确度决定输出质量
❌ 模糊：你是一位教授
✅ 精确：你是一位在[具体领域]有15年研究经验的教授，研究方向是[X]，擅长[Y]研究方法，已发表50+篇SSCI/CSSCI论文
原则：给AI的"人设"越具体，输出越专业。
技巧2：分层提问法
不要一次性让AI写一整章。正确做法：第一轮生成框架 → 第二轮逐小节生成 → 第三轮整体润色和逻辑检查。每轮质量都比一轮搞定高很多。
技巧3：提供"上下文锚点"
AI不知道你的论文整体在做什么，每次提问先给上下文：
我的论文题目是[X]，研究方法是[Y]，主要数据来源是[Z]。目前已完成第1-3章，现在要写第4章[章节名]。这一章的目的：[一句话]
技巧4：使用"思维链"引导分析
在需要AI做深层分析时加一句：请一步步分析，先……再……最后…… ——这会让AI展现推理过程，而不是直接给结论。
技巧5：反向质控
拿到AI输出后用这个提示词质控：请以一位严格审稿人的身份，批判性地审视以下内容：1.哪些论证不够充分？2.哪些地方有逻辑跳跃？3.有没有学术硬伤？4.如果是你，会在哪些地方追问？
