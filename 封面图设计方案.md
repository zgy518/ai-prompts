# 封面图设计方案

> 面包多支持上传封面图，也可使用其内置 AI 封面生成。以下是设计思路，可以用 Canva 免费制作或交给 AI 生图工具。

---

## 方案一：Canva 免费模板（推荐，10分钟搞定）

### 规格
- 尺寸：800×600 px（面包多推荐尺寸）
- 格式：JPG 或 PNG
- 大小：< 5MB

### 设计要素

**配色**：
- 主色：深蓝渐变（#1a1a2e → #16213e）—— 科技感 + 专业感
- 点缀色：荧光绿（#00ff88）—— 突出重点信息
- 文字色：白色（#ffffff）+ 浅灰（#e0e0e0）

**布局**：
```
┌──────────────────────────────┐
│                              │
│     🧠  AI 提示词            │
│     从入门到精通             │
│                              │
│     ┌─────────────────┐     │
│     │  写作·编程       │     │
│     │  学习·办公       │     │
│     │  200条精选       │     │
│     └─────────────────┘     │
│                              │
│     ChatGPT · Claude         │
│     Kimi · 通义千问 通用     │
│                              │
│     首发价 ¥9.9              │
│                              │
└──────────────────────────────┘
```

### Canva 操作步骤
1. 打开 canva.com，搜索"电子书封面"模板
2. 选择一个深色科技风的模板
3. 替换文字和配色为上述方案
4. 右上角加一个红色/橙色徽章："限时 ¥9.9"
5. 导出为 JPG，800×600 px

## 方案三：纯文字排版（最省事）

如果不想做图，直接用这个文字排版，白底黑字/黑底白字即可：

### 白底版
- 白色背景 (#ffffff)
- 顶部大号粗体：「200个AI提示词」 （深蓝 #1a1a2e，字号 48pt）
- 中间副标题：「从入门到精通」 （灰色 #666，字号 28pt）
- 底部一行小字：「ChatGPT · Claude · Kimi · 通义千问 通用」 （浅灰 #999，字号 14pt）
- 右下角红色标签：「首发 ¥9.9」

用 PPT 或 Keynote 都能做，导出为 PNG 即用。

## 封面图避坑提醒

1. **不要用网络随便找的图**——可能有版权问题，被举报会下架
2. **文字要清晰**——面包多列表页封面图很小，标题要大、对比度要够
3. **不要放二维码**——面包多禁止在封面图放引流二维码
4. **不要写"最全""第一"——** 广告法禁用词，有被投诉风险
5. **推荐用面包多内置 AI 封面生成**——零门槛，风格统一，没有版权问题
