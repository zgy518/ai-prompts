### 学习

**7. 费曼学习法**
```
请用费曼学习法帮我理解[概念]：
- 我的水平：[零基础/有一定了解]
- 要求：
  1. 用最简单的话解释（像讲给12岁小孩）
  2. 用生活类比帮助理解
  3. 举一个正例和一个反例
  4. 出一道题测试我是否真懂了
```

**8. 论文精读**
```
请帮我精读这篇论文：
- 摘要/全文：[粘贴]
- 要求：
  1. 核心贡献：这篇论文到底提出了什么新东西？
  2. 方法：用了什么方法/实验？
  3. 局限：作者承认的不足和你的怀疑
  4. 启发：对我有什么启发？
```
