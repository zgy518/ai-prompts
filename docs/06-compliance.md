## 隐私政策模板 (Privacy Policy)

```markdown
# 隐私政策

最后更新日期：[日期]

## 我们收集的信息
本网站为静态网站，不主动收集任何个人身份信息。

### 自动收集的信息
- 浏览器类型和版本
- 访问时间和日期
- 访问的页面
- 来源网站（Referrer）
（以上由 GitHub Pages 服务器日志默认记录，我们不会查看或使用。）

### 第三方服务
本网站可能使用以下第三方服务，请参阅其隐私政策：
- Gumroad（支付处理）：https://gumroad.com/privacy
- Buy Me a Coffee（赞助）：https://www.buymeacoffee.com/privacy

## Cookie
本网站不使用 Cookie。第三方服务可能使用其自身的 Cookie。

## 数据存储
本网站不在服务器端存储任何用户数据。

## 联系我们
如有隐私相关问题，请联系：[你的邮箱]
```
