# 00 — 项目总览与目标

## 项目名称（待定）
轻量变现网站

## 一句话描述
用 AI 辅助，在 GitHub Pages 上搭建一个能赚钱的静态网站。

## 目标
- **短期目标（1个月）：** 网站上线，第一款数字产品在 Gumroad 上架
- **中期目标（3个月）：** 月访问量 1000+，产生第一笔收入
- **长期目标（6个月）：** 月收入稳定在 $100+，有 2-3 款产品

## 目标用户
- 独立开发者
- 刚开始学习编程的新手
- 对 AI 工具感兴趣的技术人员

## 核心指标（KPI）
| 指标 | 1个月目标 | 3个月目标 | 6个月目标 |
|------|----------|----------|----------|
| 网站上线 | ✅ | - | - |
| 月独立访客 | 100 | 1000 | 3000+ |
| 产品数量 | 1 | 2 | 3+ |
| 月收入 | $0 | $1-$50 | $100+ |
| Gumroad 转化率 | - | 1%+ | 2%+ |
